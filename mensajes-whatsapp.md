# Mensajes de WhatsApp para Vender

Estos mensajes estan pensados para copiar, pegar y adaptar rapido.

## 1. Primer mensaje para paginas web

Hola, soy de Grupo Omega. Estamos ayudando negocios a tener una pagina web profesional para verse mejor, aparecer mas serios y recibir clientes por WhatsApp o redes. Ya tengo una demo lista y te la puedo mostrar sin compromiso. Si te interesa, te la adapto a tu negocio.

## 2. Primer mensaje para POS

Hola, soy de Grupo Omega. Estamos mostrando un sistema POS para negocios que necesitan vender con mas orden, controlar inventario y trabajar mejor en caja. Ya tengo una demo funcional lista para ensenarte. Si quieres, te la muestro y te explico como podria adaptarse a tu negocio.

## 3. Mensaje corto para enviar despues de aceptar una demo

Perfecto. Te comparto el enlace de la demo para que la revises con calma. Si te gusta, te explico como se adaptaria a tu negocio, cuanto tardaria y el costo base para arrancar.

## 4. Mensaje de seguimiento si no responde

Hola, te escribo de nuevo por si no viste mi mensaje anterior. Tengo lista una demo web y una demo de POS que podrian servirte para vender mejor y trabajar con mas orden. Si quieres, te la comparto por aqui y la ves sin compromiso.

## 5. Mensaje despues de una reunion o demo

Gracias por tu tiempo. Como te mostre, si avanzamos puedo dejarte una solucion adaptada a tu negocio. Para iniciar trabajamos con anticipo y luego cerramos el resto al entregar. Si quieres, te envio la propuesta base y los pasos para arrancar.

## 6. Mensaje para cerrar una pagina web

Si quieres avanzar, el siguiente paso es confirmar el tipo de pagina, enviarme logo, colores, servicios y datos de contacto. Trabajamos con 50 por ciento para iniciar y 50 por ciento al entregar.

## 7. Mensaje para cerrar un POS

Si quieres avanzar con el POS, definimos primero si lo quieres como instalacion local o como piloto beta. Para iniciar trabajamos con 70 por ciento de anticipo y 30 por ciento al dejarlo operando.

## 8. Variables para personalizar

- `[NOMBRE]`
- `[NEGOCIO]`
- `[TIPO DE SERVICIO]`
- `[PRECIO BASE]`
- `[TIEMPO ESTIMADO]`
- `[ENLACE DEMO]`
